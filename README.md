# Truco Series Argentina 2026 — Propuesta Técnica

Propuesta técnica de ABN Digital para el sistema de métricas y estadísticas del torneo Truco Series Argentina 2026.

## Sitio publicado

La propuesta está disponible como sitio estático en GitHub Pages:

- **Propuesta completa:** [index.html](./index.html)
- **Diseño de mesa:** [design-mesa.html](./design-mesa.html)

## Contenido

1. Resumen ejecutivo con distinción garantizado vs. sujeto a viabilidad
2. Contexto del torneo y modelo de captura de datos
3. Alcance detallado (app veedor, planillas, stats, cardcams, Monte Carlo)
4. Arquitectura de datos
5. Fases de desarrollo
6. Requerimientos del cliente (incluye video de prueba)
7. Plan de contingencia
8. Próximos pasos

## Desarrollo local

Abrir `index.html` en el navegador o servir con:

```bash
python3 -m http.server 8080
```

Documento preparado para TSA Sports. Uso interno.
